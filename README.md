A simple web scraping tool.

```
usage: scrape.py [-h] [--version] [-l http://mywebsite] [-a] [-m NUMBER]

options:
  -h, --help           show this help message and exit
  --version            show program version number and exit
  -l http://mywebsite  list all the links inside a website
  -m NUMBER            drill down to a maxdepth of

        ----------------------------------------------------------------
        This is designed to scrape websites and return all the links <a href="http://website"></a> from the page
```
